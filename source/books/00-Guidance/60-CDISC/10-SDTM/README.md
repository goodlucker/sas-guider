# SDTM

SDTM 是 CDISC 的基础标准，用于结构化呈现临床试验原始数据，已被主要监管机构广泛采纳，用于提高申报数据的一致性与可审查性。

## 数据域分类（SDTM IG）
- Trial Design（试验设计）：TA、TE、TI、TV、TS
- Special Purpose（特殊用途）：DM、SE、SV、CO
- Events（事件）：AE、MH、DS
- Interventions（干预）：CM、EX、SU、PR
- Findings（检查/检测）：LB、VS、EG、PE、QS
- Findings About（关于发现）：FA（用于记录对某一 Finding 的附加属性）
 - Relationship（关系）：RELREC（用于记录跨域或跨记录的关联关系）
- Relationship（关系）：Relrec（用于记录数据关联）
