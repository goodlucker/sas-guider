# Define

Define-XML 用于描述 SDTM/ADaM 数据集的结构与元数据（变量、取值、来源、衍生逻辑、受控术语），实现机器可读与可追溯的交付文件，是监管申报的关键组成部分。

## 项目要点
- 与 SDTM/ADaM 同步版本管理，保持标准与受控术语一致。
- 明确变量来源与衍生方法，确保与分析程序逻辑一致。
- 使用校验工具检查一致性与完整性，并根据监管机构最新指南更新。
