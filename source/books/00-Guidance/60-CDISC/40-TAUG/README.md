# TAUG

TAUG（Therapeutic Area User Guides）提供治疗领域的实施指导，帮助在特定疾病领域下正确应用 SDTM/ADaM/Define-XML 与受控术语。TAUG 通常覆盖特定域的扩展、变量使用约定、常见数据场景示例与质量检查要点。

## 使用示例
- 量表与问卷（QS）：参考相应 TAUG 选择合适的受控术语与参数编码，确保题项、得分、时间点与访视逻辑一致。
- 肿瘤评估（RS/--RS）：依 TAUG 定义响应评估结构与结果值（如 CR、PR、SD、PD），并在 RELREC 记录影像与评估的关联。
- 设备与程序（PR/DEV）：遵循 TAUG 的数据元素选择与单位规范，统一记录设备型号、程序步骤与术后评估。
