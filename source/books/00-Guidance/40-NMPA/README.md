# NMPA/CDE 指南与数据标准

- 概览：国家药品监督管理局（NMPA）负责药品、医疗器械与化妆品监管；药审中心（CDE）负责技术审评与指南发布。
- 数据与电子申报：国家正在推进数据标准化与电子申报实践，项目实施需遵循最新的政策文件与技术指南。

## 与数据标准相关的方向
- 电子申报（eSubmission）：按公告与技术要求准备申报资料与数据文件。
- 临床试验数据标准化：企业普遍采用 CDISC（SDTM/ADaM/Define-XML）以提升数据质量与国际互认。
- 统计与编程：确保数据处理可追溯、分析可复现，并符合伦理与质量管理要求。
