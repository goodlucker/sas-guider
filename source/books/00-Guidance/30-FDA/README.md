# FDA 指南与数据标准

- 概览：美国食品药品监督管理局（FDA）负责药品、生物制品与医疗器械的监管与审批。
- 数据标准：FDA 推荐并接受 CDISC 标准（SDTM、ADaM、Define-XML），并发布技术一致性指南以指导数据提交。
- 电子申报：药品注册采用电子通道与技术要求，需符合相关数据与技术规范。

## 与数据标准相关的核心资源
- Study Data Standards Catalog：列出 CDER/CBER 当前接受的 SDTM、ADaM、Define-XML 等版本。
- Study Data Technical Conformance Guide：数据提交的技术一致性指南，涵盖 SDTM/ADaM 的实施与检核要求。
- Data Standards Resources：数据标准资源总览页，包含各项技术与政策链接。

## CDISC 与 FDA
- FDA 在新药申报中广泛接受 CDISC 标准；建议在项目初期确定所用标准版本并与 Catalog 对齐。
- SDTM 技术规范与实施指南详见下方“快速链接”。
