# EMA 指南与数据标准

- 概览：欧洲药品管理局（EMA）负责欧盟范围内药品监管与科学评估，协调各成员国的药品审批。
- 数据标准与电子申报：EMA 支持电子申报与标准化数据提交，普遍采用 CDISC 标准用于临床数据的结构化与分析。

## 与数据标准相关的方向
- eSubmission 与技术要求：按官方资源准备提交包、元数据与技术文件。
- CDISC（SDTM/ADaM/Define-XML）：在临床数据提交中广泛应用，提升跨机构审评效率与一致性。
