# Encoding 概览与链接

- 概览：SAS 的编码设置影响数据读取、处理与输出的字符集与字符编码，需与数据源与目标文档一致。

## 常用任务
- 设置会话编码与地区选项（NLS），确保中文等多语言内容正确处理与导出。
- 导入/导出文本时指定编码，避免乱码与字符丢失。
- 在 ODS 输出（RTF/HTML/Excel）中保持编码一致性。
