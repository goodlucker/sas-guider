# Function 概览与链接

- 概览：SAS 函数覆盖字符串、日期时间、数学统计、正则表达式等常见处理需求，是 `DATA STEP` 与宏编程的基础能力。

## 常用分类
- 字符处理：`SUBSTR`、`SCAN`、`FINDW`、`COMPBL`。
- 日期时间：`DATE`、`TODAY`、`DHMS`、`INTNX`、`INTCK`。
- 数值统计：`ROUND`、`SUM`、`MEAN`、`STD`。
- 正则表达式（PRX）：`PRXPARSE`、`PRXMATCH`、`PRXCHANGE`。
