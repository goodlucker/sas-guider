# Macro 概览与链接

- 概览：SAS 宏语言用于参数化与自动化，支持条件与循环控制、文本替换、跨步骤变量共享，是提升复用与可维护性的关键。

## 常用要点
- 宏变量与作用域：`%LET`、`CALL SYMPUTX`、`&name`。
- 条件与循环：`%IF/%THEN/%ELSE`、`%DO/%END`、`%DO i=...`。
- 运行时函数：`%SYSFUNC` 在宏中调用数据步函数。
- 调试与日志：`OPTIONS MPRINT SYMBOLGEN MLOGIC;`。
