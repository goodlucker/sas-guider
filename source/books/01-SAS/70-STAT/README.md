# STAT 概览与链接

- 概览：SAS/STAT 提供从描述统计、假设检验到建模与生存分析的完整方法库，适合临床试验与观察性研究的数据分析。

## 常用过程
- 描述统计：`PROC MEANS`、`PROC UNIVARIATE`、`PROC FREQ`。
- 回归与建模：`PROC REG`、`PROC GLM`、`PROC MIXED`、`PROC LOGISTIC`、`PROC GENMOD`。
- 生存分析：`PROC LIFETEST`、`PROC PHREG`。
