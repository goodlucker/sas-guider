# Perl Regular Expression 概览与链接

- 概览：SAS 通过 PRX 系列函数支持 Perl 风格正则表达式，适用于复杂文本匹配与替换（如清洗原始数据、标准化编码）。

## 常用任务
- 定义模式：`PRXPARSE` 编译正则表达式。
- 匹配检测：`PRXMATCH` 判断文本是否符合模式。
- 文本替换：`PRXCHANGE` 进行批量替换或重写。
