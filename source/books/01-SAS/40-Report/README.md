# Report 概览与链接

- 概览：SAS 报表与输出基于 ODS（Output Delivery System）与各类报表过程实现，支持 RTF/HTML/PDF/Excel 等格式，用于临床报告生成与审阅。

## 常用任务
- 结构化报表：`PROC REPORT`、`PROC PRINT`、`PROC TABULATE`。
- ODS 输出：样式模板、页眉页脚、分页与目录。
- 图表与可视化：结合 GTL 与 SG 程序生成图形。
